# NotebookLM Deep Dives

Deep-dive explorations of research papers, books, and articles, generated with NotebookLM. Each episode is a conversational unpacking of a single work — or a thematic pairing of two — covering the thesis, the named technical contributions, and the open questions, in roughly the length of a commute.

The current run is a thread on embodied AI: why intelligence needs a body, how simulators and robots are getting them, how humans respond to machines that seem human, and what it means to build agents that genuinely cooperate.

## Subscribe

The show is not listed in public podcast directories. Add it by RSS feed URL:

```
https://synthyclaw.github.io/podpub/feed.xml
```

### Apple Podcasts
- **iPhone / iPad**: open the Podcasts app → **Library** tab → tap **Edit** (top right) → **Add a Show by URL…** → paste the URL → **Follow**.
- **Mac**: open Podcasts → **File** menu → **Add a Show by URL…** → paste the URL → **Follow**.

### Pocket Casts (iOS, Android, Web)
Open the **Discover** (or search) screen, paste the RSS URL into the search field, tap the podcast that appears, and tap **Subscribe**. Same flow on mobile and on the web player.

### Overcast (iOS)
Overcast adds non-directory feeds through its web form. In a browser, go to [overcast.fm/add](https://overcast.fm/add), paste the URL, and hit **Add Podcast**. It syncs to the Overcast iOS app as long as you are signed in with the same account.

---

Cover art, `feed.xml`, and all episode audio are served from this repo via GitHub Pages. New episodes appear within about a minute of a push; podcast apps may take up to an hour to refresh on their own, or immediately if you pull down to refresh in the app.
